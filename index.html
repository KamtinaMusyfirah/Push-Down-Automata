Chomsky Hierarchy

Type 0 : Unrestricted Grammar (UG)

Dalam teori automata. class Unrestricted Grammar adalah class grammar yang paling umum dalam hierarchy chomsky. Tidak ada batasan yang dibuat pada aturan produksi type 0 ini, selain setiap sisi kirinya yang tidak kosong. grammar class ini dapat menghasilkan bahasa yang dapat dihitung secara recursively enumerable.

M = = (Q, Σ, Γ, δ, q0, B, F)

Q = the set of internal states

Σ = the input symbol

Γ = a finite set of symbols called the tape symbols

δ = the transaction function

q0 = initial state

B = the blank symbols

F = set of final or accepting state

Aturan Produksi type 0 ini yaitu :

α → β

ciri : α,β ∈ (VT | VN )*, |α| > 0

Sebagai contoh,

Sab → ba
A → S

Type 1 : Context Sensitive Grammar (CSG)

Context Sensitive Grammar ( CSG ) adalah formal grammar di mana sisi kiri dan sisi kanan dari setiap aturan produksi dapat dikelilingi oleh konteks terminal dan simbol nonterminal. grammar ini lebih umum daripada context free grammar, dalam arti bahwa ada bahasa yang dapat dijelaskan oleh CSG tetapi tidak dengan Context Free Grammar. Grammar ini kurang umum (dalam arti yang sama) jika dibandingkan Unrestricted grammar. Dengan demikian, CSG diposisikan di antara context free grammar dan Unrestricted grammar dalam hierarki Chomsky.

P = (Q, Σ, Γ, δ, q0, Z0, F)

Q = the set of internal states

Σ = the input symbol

Γ = a finite set of symbols called the stack symbols

δ = the transaction function

q0 = initial state

Z0 = start symbols

F = set of final state

Aturan Produksi type 1 ini yaitu :

α → β

ciri : α,β ∈ (VT | VN )*, 0 < |α| ≤ |β|

Sebagai contoh,

S → AB
AB → abc

Type 2 : Context Free Grammar (CSG)

Bahasa yang dihasilkan oleh context free grammar ini dikenal dengan Push down automata. Produksi sisi kiri hanya dapat memiliki satu variabel. Adapun disisi kanannya tidak ada batasan. Context free grammar atau lebih tepatnya subset dari deterministic context-free language adalah dasar teoretis untuk struktur frase dari sebagian besar bahasa pemrograman , meskipun sintaksnya juga mencakup name resolution yang sensitive konteks karena deklarasi dan cakupan. Seringkali grammar ini digunakan untuk mempermudah penguraian.

G = (V, T, P, S)

Q = set of variabels

T = terminal

P = set of productions

S = start symbols

Aturan Produksi type 2 ini yaitu :

α → β

ciri : α ∈ VN ,β ∈ (VT | VN )*

Sebagai contoh,

S → A
A → a

Type 3 : Reguler Grammar (RG)

Type 3 ini adalah bentuk grammar yang paling terbatas. Reguler grammar membatasi aturannya pada satu nonterminal di sisi kiri dan sisi kanan yang terdiri dari satu terminal, memungkinkan untuk diikuti oleh satu nonterminal disebelah kanan juga. Reguler grammar ini biasanya digunakan untuk menentukan pola pencarian dan struktur leksikal bahasa pemrograman.

Aturan Produksi type 3 ini yaitu :

α → β

ciri : α ∈ VN ,β ∈ {VT, VT VN} atau α ∈ VN ,β ∈ {VT, VN VT}

Sebagai contoh,

S → aA
A → Sb
